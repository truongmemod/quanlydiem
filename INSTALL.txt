📌 YÊU CẦU:
- XAMPP (hoặc WAMP/LAMP)
- Trình duyệt web

🚀 CÀI ĐẶT NHANH (3 BƯỚC):

[BƯỚC 1] CÀI ĐẶT XAMPP
   1. Tải XAMPP: https://www.apachefriends.org/
   2. Cài đặt XAMPP
   3. Mở XAMPP Control Panel
   4. Nhấn "Start" cho Apache và MySQL

[BƯỚC 2] IMPORT DATABASE
   1. Mở trình duyệt, truy cập: http://localhost/phpmyadmin
   2. Click tab "SQL"
   3. Mở file "database.sql" trong thư mục dự án
   4. Copy toàn bộ nội dung
   5. Paste vào ô SQL trong phpMyAdmin
   6. Click "Go"
   ✅ Database đã được tạo!

[BƯỚC 3] CÀI ĐẶT SOURCE CODE
   1. Giải nén thư mục dự án
   2. Copy toàn bộ vào: C:\xampp\htdocs\quan_ly_diem
   3. Mở trình duyệt
   4. Truy cập: http://localhost/quan_ly_diem
   ✅ Hệ thống sẵn sàng!
